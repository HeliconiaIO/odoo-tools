No Mail Server Copied

Features

Duplication of database

Worried ! when duplicating the database about incoming and outgoing server !

Backup and restoration of database

After restoration of the database, don't need to execute query for removing email server data.

How the it works?

Configuration

In the Odoo configruation file, just need to add module name in the odoo-server.conf file, with server_wide_modules as shown in right side block.

If you are running odoo from terminal, you can use --load option with odoo-bin executable.

No Mail Server Info

When duplicating / backing up the database, there will be one check box for not coping the incoming and outgoing email server records.
